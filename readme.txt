This plugin creates a cool, highly customisable text widget that also serves as a light weight call to action element.

== Description ==

This plugin is both, a text widget and a call to action widget for your WordPress site. It gives site owners the ability to direct traffic to specific areas of the website using only text widgets. This plugin was specifically created with inbound marketing best practices in mind and will help you direct the users to areas of the site & convert more leads on your site.

Major features include:

* Over 30 customisation options to control the look & feel of the widget
* Inherits theme font
* Color chooser
* Display an image in the text widget
* Image chooser using default Wordpress media library
* and, several finer customisation options

== Installation ==

* Upload this plugin to your blog, activate it.
* You'll find the Kodio Text Widget in the widgets area
* Drag and drop the widget on the sidebar
* Customise the widget to match your site theme
* That's it!
